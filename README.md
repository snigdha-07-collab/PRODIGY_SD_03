# Contact Management System

## Overview
This project is a Contact Management System developed using Python and Tkinter. It provides a simple graphical user interface (GUI) that allows users to manage contact information efficiently.

## Features
- Add new contacts
- View contact details
- Search contacts by name
- Delete contacts
- User-friendly GUI
- Simple and efficient contact management

## Technologies Used
- Python
- Tkinter

## How to Run
1. Install Python on your system.
2. Download or clone this repository.
3. Open the project folder in VS Code.
4. Run the following command:

```bash
python contact_manager.py
```
